# vue3-shop

本项目使用 vue3 + vue-router + pinia + vant + axios 搭建，页面参考[华为商城](https://m.vmall.com)，接口数据也来自商城。

## 项目启动

可以通过下载或者克隆仓库代码，安装依赖，并启动项目。

``` shell
$ git clone git@github.com:LeSup/vue3-shop.git
$ cd vue3-shop
$ npm install
$ npm run dev
```
